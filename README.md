# Eloquent Composite Key extension
## To use:

```
To use this library, place the following in your composer.json file:
    "require": {
        "brestrepo/eloquent-composite": "dev-master"
    },
    "repositories": [
        {
            "type": "git",
            "url": "ssh://git@bitbucket.org/brestrepo/eloquent-composite.git"
        }
    ]
```
